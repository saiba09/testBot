EY CHAT APPLICATION

Project contains a chatbot for users along with administrator console for management. 

Getting Started



Prerequisites

--> Java SDK 1.7
--> My Sql 5.7
--> Maven 3.3
--Apache Tomcat 1.8


Installing

Setup My Sql , Tomcat and Java 
Run application on server

Packaging structure is as follows:
